REQUIREMENTS:
1. To perform read operation for Address Service

ENTITY:
1. Address
2. AddressService
3. AppException
4. ErrorCode

METHOD SIGNATURE:
1. public Address read(int id);

WORK TO BE DONE:
1. Check whether the id is invalid
    1.1 Throw AppException with message "id is invalid"
2. Establish connection using JDBC driver and store it in con of type Connection
3. Prepare select query where the id is the input id and store it in selectQuery of type String
4. Prepare "ps" using con and selectQuery
5. Execute ps
6. Return address

PSEUDOCODE:

    class AddressService {

        public Address read(int id) {
            
            if (address.id <= 0) {
                throw new AppException(errorCode, "id is invalid");
            }
            
            String url;
            String username;
            String password;

            Address address;

            try {
                Connection con = DriverManager.getConnection(url, username, password);

                String selectQuery = "SELECT street, city, postalCode FROM address WHERE id = id";
                PreparedStatement ps = con.prepareStatement(selectQuery);
                
                ResultSet rs = ps.executeQuery(); 
                address = new Address(rs.getString(street), rs.getString(city), rs.getInt(postalCode));
            } catch() {
                System.out.println(errorCode, errorMessage);
            }

            return address;
        }
    }
