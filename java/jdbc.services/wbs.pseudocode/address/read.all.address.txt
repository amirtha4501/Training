REQUIREMENTS:
1. To perform read all operation for Address Service

ENTITY:
1. Address
2. AddressService
3. AppException
4. ErrorCode

METHOD SIGNATURE:
1. public List<Address> readAll();

WORK TO BE DONE:
1. Create a list of type Address
2. Establish connection using JDBC driver and store it in con of type Connection
3. Prepare select query to query all addresses and store it in selectQuery of type String
4. Prepare "ps" using con and selectQuery
5. Execute ps
6. While rs has rows, add the rows to the addresses list
7. return address list

PSEUDOCODE:

    class AddressService {

        public List<Address> readAll() {
            
            String url;
            String username;
            String password;

            List<Address> addresses = new ArrayList<>();

            try {
                Connection con = DriverManager.getConnection(url, username, password);

                String selectQuery = "SELECT street, city, postalCode FROM address";
                PreparedStatement ps = con.prepareStatement(selectQuery);
                
                ResultSet rs = ps.executeQuery(); 

                while (rs.next()) {
                    addresses.add(new Address(rs.getString(street) + rs.getString(city) + rs.getInt(postalCode)));
                }
            } catch() {
                System.out.println(errorCode, errorMessage);
            }

            return addresses;
        }
    }
