REQUIREMENTS:
1. To perform read all operation for Person Service

ENTITY:
1. Person
2. PersonService
3. AppException
4. ErrorCode

METHOD SIGNATURE:
1. public List<Person> readAll();

WORK TO BE DONE:
1. Create a list of type Person
2. Establish connection using JDBC driver and store it in con of type Connection
3. Prepare select query to query all persons and store it in selectQuery of type String
4. Prepare "ps" using con and selectQuery
5. Execute ps
6. While rs has rows, add the rows to the persons list
7. return Person list

PSEUDOCODE:

    class PersonService {

        public List<Person> readAll() {
            
            String url;
            String username;
            String password;

            List<Person> persons = new ArrayList<>();

            try {
                Connection con = DriverManager.getConnection(url, username, password);

                String selectQuery = "SELECT name, email, address_id, birth_date, created_date FROM person";
                PreparedStatement ps = con.prepareStatement(selectQuery);
                
                ResultSet rs = ps.executeQuery(); 

                while (rs.next()) {
                    persons.add(new Person(rs.getString(name) + rs.getString(email) + rs.getInt(address_id) + + rs.getString(birth_date) + + rs.getString(created_date)));
                }
            } catch() {
                System.out.println(errorCode, errorMessage);
            }

            return persons;
        }
    }
