REQUIREMENTS:
1. To perform read operation for Person Service

ENTITY:
1. Person
2. PersonService
3. AppException
4. ErrorCode

METHOD SIGNATURE:
1. public Person read(int id);

WORK TO BE DONE:
1. Check whether the id is valid
    1.1 Throw AppException with message "id is invalid"
2. Establish connection using JDBC driver and store it in con of type Connection
3. Prepare select query where the id is the input id and store it in selectQuery of type String
4. Prepare "ps" using con and selectQuery
5. Execute ps
6. Return person

PSEUDOCODE:

    class PersonService {

        public Person read(int id) {
            
            if (person.id <= 0) {
                throw new AppException(errorCode, "id is invalid");
            }
            
            String url;
            String username;
            String password;

            Person person;

            try {
                Connection con = DriverManager.getConnection(url, username, password);

                String selectQuery = "SELECT name, email, address_id, birth_date, created_date FROM person WHERE id = id";
                PreparedStatement ps = con.prepareStatement(selectQuery);
                
                ResultSet rs = ps.executeQuery(); 
                person = new Person(rs.getString(name), rs.getString(email), rs.getInt(address_id), rs.getString(birth_date), rs.getString(created_date));
            } catch() {
                System.out.println(errorCode, errorMessage);
            }

            return person;
        }
    }
